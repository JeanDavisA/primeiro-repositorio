Linha de Comandos

$ ls - listar
$ ls -a - lista arquivos ocultos
$ cd - trocar de diretório, trocar de pasta
$ cd .. - voltar a um nível no diretório
$ mkdir - criar uma pasta
$ echo - efeito mudança no arquivo ou nome/texto
$ git init - iniciará um repositório vazio .git ou usa durante a configuração inicial de um novo repositório.
$ git add * ou . - adiciona uma alteração no diretório ativo à área de staging. Ele diz ao Git que você quer incluir atualizações a um arquivo específico no próximo commit.
$ git config - usada para definir valores de configuração do Git em projetos de nível global ou local. Modifica arquivos de texto de configuração.
$ git commit - alterações feitas no repositório, escrever uma mensage m que explique a alteração no repositório.
$ git push - é usado para enviar o conteúdo do repositório local para um repositório remoto.
$ git pull - é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.
